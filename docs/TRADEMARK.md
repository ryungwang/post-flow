# 상표 등록 계획 — "Synub" (하우스 마크)

> **전략**: 제품별 브랜드(Bloomcast 등) 대신 **"Synub" 하나를 등록**해 office·billing·center·소셜 제품 전체를 덮는다.
> **근거**: 법인 등기 ≠ 상표권. 한국은 **선출원주의**(먼저 낸 사람이 임자)라 비어있을 때 빨리 잡아야 함.
> **확인 결과(KIPRIS)**: "Synub" 국내 **0건**(완전 비어있음), 해외 1건은 오탐(GRANDNADOR 와인, 무관).

## 지정상품 (특허로 "지정상품 검색"에서 표준 고시명칭으로 선택)

### 🔴 9류 — 다운로드 앱 (앱 하니까 필수)
```
컴퓨터 소프트웨어(다운로드 가능한 것)
모바일 애플리케이션 소프트웨어(다운로드 가능한 것)
응용 소프트웨어(다운로드 가능한 것)
```

### 🔴 42류 — 웹/SaaS/클라우드 (웹 하니까 필수)
```
서비스형 소프트웨어(SaaS)업
온라인 비다운로드 소프트웨어 제공업
컴퓨터 소프트웨어 개발업
컴퓨터 소프트웨어 유지·보수업
클라우드 컴퓨팅 서비스업
컴퓨터 소프트웨어 디자인업
```

### 🟡 35류 — 광고·마케팅 (소셜 자동화 제품이라 권장)
```
온라인 광고업
광고 대행업
소셜미디어 마케팅 대행업
사업정보 제공업
```

## 비용 & 우선순위
| 조합 | 출원료 | 커버 |
|---|---|---|
| **9 + 42** (웹+앱) | ~11.2만원 | 🔴 최소 필수 |
| 9 + 42 + 35 | ~16.8만원 | 🟢 마케팅까지 |

- 1류당 약 5.6만원, **류마다 따로** 납부
- **출원료만 지금**, 등록료(~21만원/류)는 승인(~1년 뒤) 시 납부
- 같은 류 안 지정상품 **20개 이내면 추가비용 없음** → 위 문구 다 넣어도 OK

## 팁 (수수료·거절 방지)
- 반드시 **KIPO 고시명칭(표준 문구)으로 선택** → 기본 수수료 유지 + 거절 위험↓. 직접 타이핑한 비표준 문구는 추가비용·심사 까다로움.
- **문자상표**로 출원 → 로고 스타일 바꿔도 이름 보호(제일 넓음).
- 출원일 = **선출원 순번 확보**. 하루라도 빨리.

## 참고: 네이밍·도메인 결정
- **제품 도메인** = `<제품>.synub.io` 서브도메인 (예: `social.synub.io`) → 별도 TLD 구매 불필요, 도메인 충돌 회피.
- **PostFlow**: 동종(소셜툴)에 다수 존재 + `PostFlow®` 등록상표 → 브랜드로 부적합.
- **Bloomcast**: 국내 42류 소멸(가능)이나 **일본 42류(Syngenta)·미국 BloomCast Inc.** 충돌 → 글로벌 부적합.
- → **"Synub" 하우스 마크 + 제품은 서브브랜드(Synub Social 등)** 전략 채택.
