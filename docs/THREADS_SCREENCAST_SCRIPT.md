# Threads 검수 스크린캐스트 대본 (한 번에 6권한)

> **한 번의 화면 녹화**로 6개 권한 클립을 쭉 담고, 나중에 구간별로 잘라 업로드.
> 총 3~4분. 테스터 실계정(`@haru_.developer`)으로.

## 준비
- 브라우저 탭 A: `https://postflow.synub.io` (haru로 로그인된 상태)
- 브라우저 탭 B(또는 폰): `threads.net` / Threads 앱 — **결과 확인용** (발행/삭제 증명)
- 녹화: **⌘⇧5 → "전체 화면 기록" → 기록 시작**
- 자막은 화면에 텍스트로 넣거나 음성 내레이션(영어/한국어 무방). 아래 `자막:` 문구 사용.

---

## ▶ CLIP ① content_publish (+ OAuth 필수)
> 자막: **"1) Connect Threads (OAuth) & Publish a post"**

1. 사이드바 **Threads → 계정 연결** 이동
2. **"다시 연결"** 클릭 → **Threads OAuth 동의 화면**이 뜸 *(이 장면이 검수 필수!)*
3. 동의 화면에서 요청 권한 목록이 보이는 상태로 **"@haru_.developer 계정으로 계속"** 클릭 → 연결 완료 토스트
4. 사이드바 **콘텐츠 → AI 생성** → 주제 입력(예: "생산성 팁") → **생성** 클릭 → 결과 카드 표시
5. 카드에서 **"발행"** 클릭 → "발행되었어요" 토스트 확인
6. **탭 B(threads.net)로 전환** → 방금 올라온 게시물 표시 *(실제 발행 증명)*
7. (가점) 다시 앱 → **스케줄**에서 예약 발행 화면 잠깐 보여주기

---

## ▶ CLIP ② manage_insights
> 자막: **"2) View insights of my own posts"**

1. 사이드바 **Threads → 인사이트**
2. 상단 KPI(팔로워·조회·좋아요·참여율) 보이게 잠깐 정지
3. 아래로 스크롤 → **베스트 게시물** + **요일별 평균 참여율 차트** 표시

---

## ▶ CLIP ③ read_replies
> 자막: **"3) Read replies on my posts"**

1. 사이드바 **Threads → 내 게시물**
2. 댓글이 있는 게시물의 **"댓글 N개"** 클릭해 펼치기 → 실제 댓글 목록 표시

---

## ▶ CLIP ④ manage_replies (댓글 자동화)
> 자막: **"4) Auto-reply to comments by rule"**

1. 사이드바 **Threads → 댓글 자동화**
2. **규칙 추가** → 키워드(예: "가격") + 자동 답글 문구 입력 → **저장**
3. 규칙 목록에 생성된 규칙 표시
4. (있으면) 규칙이 적용돼 **자동 답글이 달린 게시물**을 threads.net에서 확인 *(실제 동작 증명)*

---

## ▶ CLIP ⑤ manage_mentions
> 자막: **"5) View posts that mention me"**

1. 사이드바 **Threads → 멘션**
2. 나를 언급(@)한 게시물 목록 표시 → 항목 하나 "Threads에서 보기" 살짝 보여주기

---

## ▶ CLIP ⑥ delete
> 자막: **"6) Delete my own post"**

1. 사이드바 **Threads → 내 게시물**
2. CLIP①에서 발행한 게시물의 **삭제(휴지통) 버튼** 클릭 → 확인 다이얼로그 → **삭제**
3. **탭 B(threads.net) 새로고침** → 그 게시물이 사라진 것 확인 *(실제 삭제 증명)*

---

## 녹화 후
- 하나의 영상으로 6개가 다 있으면 **그대로 6개 폼에 같은 영상 업로드해도 되고**, 구간별로 잘라 각 권한에 맞는 클립만 올려도 됨.
- 각 폼: `THREADS_REVIEW_SUBMIT.md`의 **권한별 설명 붙여넣기 → 이 영상 업로드 → 동의 체크 → 저장**.
- 6개 저장 후 **검토 제출**.

## 촬영 팁
- OAuth 동의 화면은 **반드시** 포함 (CLIP①). 이거 없으면 대부분 반려.
- 각 동작은 **천천히**, 클릭 후 1~2초 정지 (리뷰어가 따라올 수 있게).
- 실계정이라 CLIP① 발행 → CLIP⑥ 삭제로 **자동 정리**됨(테스트 글 안 남음).
